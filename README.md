# Anlisis de Ventas de Videojuegos Ice Store - Proyecto
Sprint 6 Proyecto

## Descripción General
Este proyecto analiza un dataset histórico de ventas de videojuegos para identificar los factores que influyen en el éxito comercial de un título. El objetivo principal fue detectar patrones de comportamiento del mercado y generar insights accionables para planificar campañas publicitarias y priorizar plataformas y géneros prometedores para el año siguiente.

El análisis simula un caso real: estamos en diciembre de 2016 y debemos preparar la estrategia de marketing para 2017 basándonos en datos históricos.
